# Ultimate Life Tracker 🌱

A clean, single-page life tracking app for keeping everyday priorities in one place. Ultimate Life Tracker helps you manage tasks, build habits, log fitness and health activity, and keep an eye on expenses from a simple dashboard.

## ✨ Features

- 🔐 PIN-based app lock for a lightweight privacy layer
- 📊 Dashboard with quick summaries for tasks, habits, calories, and monthly expenses
- ✅ Task and study tracker with categories and completion status
- 🔥 Habit tracker with daily check-ins and streak calculation
- 💪 Fitness and health logging for calories and workout minutes
- 💸 Expense tracker with amount, category, note, and monthly total
- 🌓 Light and dark theme toggle
- 💾 Local browser storage, so your data stays on your device
- 📱 Responsive layout for desktop and smaller screens

## 🛠️ Tech Stack

- 🌐 HTML
- 🎨 CSS
- ⚡ Vanilla JavaScript
- 💾 Browser `localStorage` for persistence
- 🔑 Web Crypto API for PIN hashing when available

## 🚀 Getting Started

No build tools or package installation are required.

1. Open `index.html` in your browser.
2. Create a 4-digit passcode on first launch.
3. Log in and start adding your tasks, habits, health logs, and expenses.

## 🗄️ Data Storage

The app stores data in your browser using `localStorage` under the key:

```text
ultimateLifeTracker_appState
```

Because this is local-only storage, clearing browser data or switching browsers/devices will remove or hide the saved tracker data.

## 📝 Notes

This project is designed as a lightweight personal productivity dashboard. It does not require a backend server, account system, or database, which makes it easy to run and modify.
